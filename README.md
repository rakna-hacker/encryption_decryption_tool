# Encryption and Decryption Tool

## Overview
This project is a simple Encryption and Decryption Tool developed using Python. It allows users to encrypt plain text into cipher text and decrypt cipher text back into the original message.

## Features
- Encrypt text messages
- Decrypt encrypted messages
- User-friendly interface
- Fast and secure processing

## Technologies Used
- Python
- Tkinter (if you used GUI)
- Cryptography libraries (if applicable)
